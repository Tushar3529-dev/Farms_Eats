# 🌾 Farm Eats App

A modern Flutter-based mobile application that connects users with fresh farm produce. The app provides a seamless experience for browsing, authentication, and managing farm-related information.

---

## 🚀 Features

* 🔐 User Authentication (Login / Sign Up)
* 📩 OTP Verification & Password Reset
* 🌱 Farm Information Registration
* 🕒 Business Hours Setup
* 🏠 Clean and Responsive Home Screen
* 🎨 Modern UI with Custom Themes
* 📱 Cross-platform (Android, iOS, Web)

---

## 📸 Screens Included

* Onboarding Screens
* Login & Sign Up
* OTP Verification
* Forgot / Reset Password
* Farm Information Form
* Home Screen

---

## 🛠️ Tech Stack

* **Flutter** – UI Development
* **Dart** – Programming Language
* **REST APIs** – Backend Integration
* **MVVM-like Structure** – Clean code organization

## 📂 Project Structure

```
lib/
│
├── common/               # Reusable widgets & styles
├── features/

│   ├── home/            # Home screen
│
├── utils/               # Helpers, constants, themes
├── main.dart            # Entry point
└── my_app.dart          # App configuration
```

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/Tushar3529-dev/Farms_Eats.git
```

### 2. Navigate to project

```bash
cd Farms_Eats
```

### 3. Get dependencies

```bash
flutter pub get
```

### 4. Run the app

```bash
flutter run
```

---

## 📦 Build APK

```bash
flutter build apk --release
```
